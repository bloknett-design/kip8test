## Что нового в v30

### 4 интерактивных документа с полным текстом
- **ПУЭ 6 и 7** — 43 главы, 2862 пункта
- **ПТЭЭП 2003** — 28 глав, 703 пункта
- **ПОТЭЭ 2022** — 47 глав, 625 пунктов
- **СО 153-34.03.603-2003** — 35 глав, 394 пункта

Реализовано способом 3 (ленивая загрузка по главам):
- При открытии документа грузится только `meta.json` (~2 КБ)
- При раскрытии главы грузится только файл этой главы (~5–50 КБ)
- Загруженные главы кэшируются в памяти браузера
- После первого захода — работает офлайн (кэшируется Service Worker)

### Гиперссылки из билетов до 1000 В
- 73 ссылки из билетов стали кликабельными — ведут на конкретные пункты в 4 документах
- При клике: переход на страницу документа → раскрытие нужной главы → прокрутка к пункту с подсветкой
- Ссылки на внешние документы (ГОСТ, Приказ Роскомторга) — остались как внешние ссылки

### Навигация
- Главная → Документация → Библиотека → Электробезопасность → 4 документа
- Каждый документ: аккордеон с главами, ленивая загрузка, поиск по структуре

## Установка в репозиторий

1. Распакуйте архив в корень репозитория `kip8test`
2. Замените существующие файлы:
   - `index.html` → обновлённый
   - `sw.js` → обновлённый (CACHE_VERSION = kipia-test-v30)
3. Добавьте новые папки:
   - `data/pue/` (44 файла)
   - `data/pteep/` (29 файлов)
   - `data/pote/` (48 файлов)
   - `data/so153/` (36 файлов)
4. Закоммитьте изменения: `git add . && git commit -m "v30: 4 документа с полным текстом + гиперссылки из билетов"`
5. Запушьте: `git push`
6. Проверьте: https://bloknett-design.github.io/kip8test/
